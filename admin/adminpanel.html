Admin Panel Help & Support

Welcome to the Admin Panel Help Center. Below you'll find guides on how to use the panel's features. If you encounter issues, please use the report buttons below.

How to Use the Admin Panel:

  • Managing Products: Click on "Products" to view, edit, or add new items. Use the "Add Product" button for new entries.
  • Managing Reels: Click on "Reels" to embed Instagram content. Paste the full reel link and click "Add Reel."
  • Reporting & Data: Access reports from the sidebar (if available) to view analytics.
