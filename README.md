# Developing
#  Kirpa Home Solutions

A responsive multi-page **Product Catalog Website** built for **Kirpa Electronics & Home Appliances**, designed to streamline product browsing and management.  
Developed with **HTML**, **CSS**, **JavaScript**, and **Firebase**, this project delivers a smooth, modern, and user-focused experience with both **client-side** and **admin-side** functionality.

---

##  Features

-  **Multi-Category Catalog:**  
  Separate product sections for **Hardware**, **Paint**, and **Kitchen** stores with dedicated navigation and filters.

-  **Smart Search & Filters:**  
  Intuitive search bar and category filters for easy product discovery.

-  **Product Enquiry Form:**  
  Users can directly send enquiries for specific items — integrated with Firebase for data management.

-  **Admin Dashboard:**  
  A custom admin panel for adding, editing, and deleting products, including an **activity log** for tracking all actions.

- **Modern UI/UX Design:**  
  Designed end-to-end in **Figma**, focusing on usability, color consistency, and a professional retail look.

-  **Responsive Design:**  
  Fully optimized for desktop, tablet, and mobile devices.

---

## Tech Stack

**Frontend:** HTML5, CSS3, JavaScript  
**Database / Backend:** Firebase (Firestore + Hosting)  
**Design Tools:** Figma, Canva  
**Version Control:** Git, GitHub  
**Deployment:** Vercel
